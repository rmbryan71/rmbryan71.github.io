# The Difference Between a File and a Memory

## The Idea
A file named `npr_224814293.mp3` is useless without context. A file named `20190604_cartalk_1923_electronics.mp3` tells you exactly what it is, when it aired, and what it's about. That difference — the information attached to the content rather than the content itself — is called metadata, and it's often more valuable than the thing it describes.

## Why It's Interesting
Metadata is invisible until it's missing. Your phone's photo library is useful because it knows when and where each picture was taken. A library catalog is useful because it tells you where to find a book without you having to open every book. The Car Talk archive is a small, relatable example of a concept that underlies how databases, search engines, and digital collections work.

## Key Points to Cover
- What metadata is: information about information
- The specific example: converting ID-number filenames to descriptive ones with dates and episode names
- Why it matters: findability, context, meaning
- Everyday examples: photo EXIF data, email subject lines, music tags in a streaming app
- What happens when metadata is wrong or missing: the archive becomes an unusable pile
